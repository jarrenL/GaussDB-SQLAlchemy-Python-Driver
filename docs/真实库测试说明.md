# 真实 GaussDB 测试说明

本文用于说明本项目如何连接真实 GaussDB 执行测试，覆盖哪些测试场景，以及当前验证结果。

## 1. 测试目标

验证 `gaussdb-sqlalchemy-driver` 在真实 GaussDB 环境下是否支持：

- ODBC 方式连接 GaussDB。
- SQLAlchemy Core、ORM、事务、连接池。
- 元数据反射，包括列、主键、唯一约束、普通索引、表达式索引、视图和分区表。
- Alembic Operations 和 autogenerate 基础无差异检测。
- 常用数据类型，包括 `numeric`、`timestamp`、`date`、`boolean`、`text`、`bytea` 或目标兼容模式下的等价二进制类型。
- A 兼容、B 兼容、M 兼容下的 SQL 语法差异。

## 2. 测试环境要求

### 2.1 Python 环境

建议使用 Python 3.11 或 3.12。安装本项目 wheel 时会自动安装 `pyodbc`，无需单独安装。

推荐：

```bash
python3.12 -m venv .venv
. .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[test]"
```

### 2.2 ODBC 驱动

需要安装 GaussDB ODBC Driver，并确保系统 ODBC Driver Manager 能找到该驱动。

- **Linux**：安装 `unixODBC`，再将 GaussDB ODBC 驱动 `.so` 文件路径配置到 `/etc/odbcinst.ini`。
- **macOS**：通过 `brew install unixodbc` 安装 ODBC Driver Manager，再配置 GaussDB ODBC 驱动。
- **Windows**：从华为云控制台下载 GaussDB ODBC Driver 安装包，安装后自动注册到 ODBC 数据源管理器。

确认驱动已注册：

```bash
python -c "import pyodbc; print([d for d in pyodbc.drivers() if 'gauss' in d.lower() or 'postgres' in d.lower()])"
```

驱动名称通常为 `GaussDB ODBC Driver`，用于连接串中的 `driver` 参数。

## 3. 连接配置方式

所有真实库测试都通过环境变量配置连接串，代码里不写死数据库地址。

单库验证：

```bash
export GAUSSDB_TEST_URL='gaussdb+odbc://用户名:URL编码后的密码@数据库IP:端口/数据库名?driver=GaussDB+ODBC+Driver&sslmode=disable'
```

A/B/M 三种兼容库分别验证：

```bash
export GAUSSDB_TEST_URL_A='gaussdb+odbc://用户名:URL编码后的密码@数据库IP:端口/A兼容数据库名?driver=GaussDB+ODBC+Driver&sslmode=disable'
export GAUSSDB_TEST_URL_B='gaussdb+odbc://用户名:URL编码后的密码@数据库IP:端口/B兼容数据库名?driver=GaussDB+ODBC+Driver&sslmode=disable'
export GAUSSDB_TEST_URL_M='gaussdb+odbc://用户名:URL编码后的密码@数据库IP:端口/M兼容数据库名?driver=GaussDB+ODBC+Driver&sslmode=disable'
```

也可以一次传入多个连接串：

```bash
export GAUSSDB_TEST_URLS='gaussdb+odbc://.../db1?driver=GaussDB+ODBC+Driver&sslmode=disable,gaussdb+odbc://.../db2?driver=GaussDB+ODBC+Driver&sslmode=disable'
```

如果使用 DSN 模式（需先在 ODBC 数据源管理器或 `odbc.ini` 中配置 DSN）：

```bash
export GAUSSDB_TEST_URL='gaussdb+odbc://用户名:URL编码后的密码@/数据库名?dsn=MyGaussDB'
```

密码中如果包含 `@`，需要写成 `%40`。例如 `password@123` 应写成 `password%40123`。

## 4. 测试场景

### 4.1 环境检查

命令：

```bash
python scripts/check_windows_env.py --url "$GAUSSDB_TEST_URL"
```

验证内容：

- Python 版本。
- SQLAlchemy、项目包、pyodbc 是否可导入。
- ODBC 驱动是否已注册。
- 是否可以连接真实库并执行 `select 1`。

### 4.2 基础集成测试

命令：

```bash
pytest -m integration tests/test_integration_env.py -q
```

验证内容：

- SQLAlchemy Core 建表、插入、查询。
- 真实表生命周期：建表前存在性检查、建表后存在性检查、列反射、批量插入、条件查询、更新、删除、清理后存在性检查。
- 事务回滚。
- 批量插入。
- ORM CRUD。
- 元数据反射。
- 主键、唯一约束、普通索引反射。
- A/B 下序列和 `nextval()` 默认值，M 下 `auto_increment`。
- Alembic Operations。
- 常用数据类型。
- Alembic autogenerate 基础无差异检测。
- 复合索引、表达式索引、视图反射。
- 分区表反射。
- 连接池基础复用。

### 4.3 A/B/M 兼容场景测试

命令：

```bash
pytest -m integration tests/test_compatibility_scenarios.py -q
```

验证内容：

- 当前库 `datcompatibility` 识别。
- PostgreSQL 风格基础 SQL。
- Oracle 风格 SQL。
- MySQL 风格 SQL。
- `serial`、`auto_increment`、`nextval` 差异。
- SQLAlchemy 表达式索引在不同兼容模式下的编译和执行。

### 4.4 轻量探针

在没有 pytest 的环境，可以运行脚本探针：

```bash
python scripts/run_integration_probe.py --url "$GAUSSDB_TEST_URL"
python scripts/run_syntax_probe.py --url "$GAUSSDB_TEST_URL"
```

## 6. 兼容差异摘要

### 6.1 A 兼容

已验证通过：

- PostgreSQL 基础语法：`::` cast、`now()`、`limit`、`serial`。
- Oracle 风格基础语法：`dual`、`nvl`、`sysdate`、`rownum`。
- SQLAlchemy Core、ORM、事务、反射、Alembic 基础场景。

已验证不通过或不声明支持：

- MySQL 反引号别名。
- `ifnull()`。
- `current_timestamp()` 带括号形式。
- `auto_increment`。

### 6.2 B 兼容

已验证通过：

- PostgreSQL 基础语法：`::` cast、`now()`、`limit`、`serial`。
- Oracle 风格基础语法：`dual`、`nvl`、`sysdate`、`rownum`。
- MySQL 风格语法：反引号、`ifnull()`、`concat()`、`auto_increment`。
- SQLAlchemy Core、ORM、事务、反射、Alembic 基础场景。

已验证不通过或不声明支持：

- `current_timestamp()` 带括号形式。

### 6.3 M 兼容

已验证通过：

- MySQL 风格语法：反引号、`ifnull()`、`concat()`、`current_timestamp()`、`auto_increment`。
- 独立 `select nextval('sequence_name')`。
- SQLAlchemy Core、ORM、事务、反射、Alembic 基础场景。
- SQLAlchemy 表达式索引。方言会将 `Index(func.lower(column))` 编译为 GaussDB M 可接受的双括号表达式索引。

已验证不通过或不声明支持：

- `serial`、`bigserial`。
- `default nextval('sequence')` 作为列默认值。
- Oracle 风格 `nvl`、`sysdate`、`rownum`。

方言已做的 M 兼容适配：

- `Integer primary_key` 不自动编译成 `SERIAL`。
- `DateTime` 编译为 `TIMESTAMP`。
- `LargeBinary` 编译为 `BLOB`。
- Boolean 按 0/1 读写并返回 Python `bool`。
- 表达式索引编译为双括号形式。
